Parágrafos e Formatação de Texto

Este é um parágrafo básico, O elemento <p> é usado para criar parágrafos em HTML.

Texto em negrito pode ser criado com o elemento <strong>.

Texto em Itálico pode ser criado com o elemento <em> ou <i>.


Esse segundo bloco, por sua vez, é do segundo aluno

Essa linha, bem menor que a anterior, foi feita com <p> para criar o parágrafo

O negrito é usado para reforçar certas informações. Ele é de fato muito útil, dependendo do que você procurar fazer e, para "negritar", deve-se usar a tag <strong> ou <b>.

Já este texto poderia ser uma citação, caso fosse acadêmico. Eu o fiz com <em>, mas poderia ter usado <i>
